Consumindo API



Random fox - API que exibe fotos de raposas


JSON Placeholder - API que simula requisições POST





meowfacts - API que retorna fatos aleatórios sobre gatos



GNews API - Api para consultar notícias(utiliza chave para acessar)



Comandos para requisição via curl

curl https://randomfox.ca/floof/

curl https://meowfacts.herokuapp.com/?lang=por-br

Linux: curl -X POST https://jsonplaceholder.typicode.com/posts \ -H "Content-Type: application/json" \ -d '{"title":"Post via curl","body":"Enviado pelo terminal","userId":1}'

Windows: curl -X POST https://jsonplaceholder.typicode.com/posts -H "Content-Type: application/json" -d "{\"title\":\"Post via curl\",\"body\":\"Enviado pelo terminal\",\"userId\":1}"
